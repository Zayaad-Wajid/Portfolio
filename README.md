# Zayad Portfolio

A sleek React + Tailwind CSS portfolio for showcasing AI, GenAI, web, and full stack projects.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173`.

## Build For Deployment

```bash
npm run build
```

Deploy the generated `dist` folder, or connect this repository to Vercel, Netlify, or GitHub Pages.

## Personalize

Edit `src/App.jsx` to update:

- `profile` for your email, GitHub, LinkedIn, and location.
- `projects` for your real AI, GenAI, web, and full stack work. The current layout has five project slots ready for live Vercel links.
- `tools` for the rotating Tools And Tech boxes.
- The theme switch defaults to the user's system preference and also supports pinned light or dark mode.
